# Don't Get Kicked!

Проект по предсказанию плохой для покупки машины.
Датасет соревнования Kaggle **"Don't Get Kicked!"**.
Целевая переменная - isBadBuy

## Цель проекта
Изучение техник валидации и подбора гиперпараметров, методов отбора признаков

## Описание данных
В работе использовался стандартный датасет `train.json` из конкурса.
