# Rental Listing Price Prediction 🏠

Проект по прогнозированию стоимости аренды недвижимости.
Датасет конкурса Kaggle **"Two Sigma Connect: Rental Listing Inquiries"**.
Целевая переменная - price

## 🎯 Цель проекта
Изучение основ Supervised Learning.

## 📊 Описание данных
В работе использовался стандартный датасет `train.json` из конкурса.
- Для корректной оценки качества модели исходный набор данных был разделен на две части: **Train** и **Test**
